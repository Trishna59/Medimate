# MediMate 💊

*A Smart Medicine Management & Reminder Mobile Application*

##  Overview

MediMate is a Flutter-based mobile application designed to improve medication safety and adherence. It allows users to scan prescriptions and medicine labels using OCR, understand medicine details through text and voice, and manage their medication schedules with smart reminders. The app is especially useful for users in rural areas or those who struggle with managing multiple medications.

---

##  Features

*  OCR-based prescription & medicine label scanning
*  Medicine information display (text + voice assistance)
*  Smart medication reminders and scheduling
*  User authentication system
*  Helps prevent double dosing and missed medication
*  Simple and user-friendly mobile UI

---

## 🛠️ Technologies Used

* Flutter (Frontend Framework)
* Dart (Programming Language)
* Firebase (Authentication & Backend Services)
* OCR Technology (Text extraction from images)
